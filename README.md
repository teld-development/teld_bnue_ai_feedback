# AI 수업실연 피드백 시스템

예비교사를 위한 AI 기반 수업 영상 분석 및 피드백 시스템입니다. 수업 영상을 업로드하면 AI가 다양한 관점에서 수업을 분석하고 구체적인 피드백을 제공합니다.

---

## 📋 주요 기능

### 1. 사용자 온보딩
- 첫 로그인 시 예비교사 정보 수집 (학년, 수업 자신감, 수업 경험)
- 개인화된 피드백 제공을 위한 기초 데이터 구성

### 2. 수업 분석 준비
수업 영상 분석 전 필요한 정보를 입력합니다:
- **학년/과목 선택**: 초등학교 1~6학년, 과목 선택
- **단원명 입력**: 분석할 수업의 단원명 기입
- **지도안 업로드** (선택): 수업 지도안 파일 첨부
- **피드백 영역 선택**: 원하는 피드백 영역 다중 선택
- **추가 분석 조건** (선택): 특별히 확인받고 싶은 사항 입력

### 3. AI 수업 영상 분석
업로드된 수업 영상을 AI가 분석하여 다음 영역별 피드백을 제공합니다:

| 영역 | 분석 내용 |
|------|----------|
| 🎤 **전달력** | 목소리 크기, 발음 정확성, 말하기 속도, 핵심 개념 설명의 명확성 |
| 💬 **상호작용** | 발문 활용, 학생 참여 유도, 질의응답 처리 방식 |
| 🙂 **태도** | 눈 맞춤, 표정, 수업 분위기 형성 |
| 📋 **내용 구성** | 수업 흐름, 도입-전개-정리 구성, 성취기준 연계 |
| 📊 **판서/자료** | 시각 자료 활용, 핵심 내용 강조 방식 |
| 🔄 **수업 습관** | 반복되는 언어 습관(말버릇, 추임새), 행동 패턴 분석 |

### 4. 타임스탬프 기반 피드백
- 수업 영상의 특정 시점에 대한 구체적인 피드백 제공
- 타임스탬프 클릭 시 해당 영상 위치로 자동 이동
- 각 피드백에 카테고리 및 유형(칭찬/개선) 표시

### 5. AI 대화형 피드백
- 수업 분석 결과를 바탕으로 AI와 추가 대화 가능
- 궁금한 점이나 개선 방법에 대해 질문하고 답변 받기

### 6. 영역별 피드백 히스토리
- 대시보드에서 영역별 최근 피드백 요약 확인
- 지난 분석 결과와 비교하여 성장 추이 파악

### 7. XR 통합 대시보드
VR 기반 예비교사 실습과 오프라인 수업 실연을 통합 관리합니다:
- **VR 문제상황 실습**: 학습 동기 저하, 수업 방해 행동 대응, 학습 수준 차이 대응 등 시나리오별 피드백
- **오프라인 수업 실연**: 실제 수업에 대한 영역별 피드백 기록
- **종합 피드백 요약**: 전체 실습에 대한 강점 및 성장 영역 분석

## 📱 화면 구성

| 페이지 | 설명 |
|--------|------|
| **로그인** | 사용자 인증 화면 |
| **온보딩** | 첫 로그인 시 예비교사 정보 입력 |
| **대시보드** | 메인 화면, 영역별 최근 피드백 요약 |
| **분석 준비** | 수업 정보 입력 및 피드백 영역 선택 |
| **영상 업로드** | 수업 영상 파일 업로드 |
| **분석 결과** | AI 분석 결과 및 타임스탬프 피드백 확인 |
| **XR 통합 대시보드** | VR 실습 + 오프라인 실연 통합 피드백 |

## 🎯 피드백 체계

### 피드백 유형
- ✅ **칭찬 (Positive)**: 잘한 점, 유지해야 할 강점
- 🔧 **개선 (Constructive)**: 개선이 필요한 점과 구체적인 방법

### 성취기준 연계 (수학 과목)
수학 과목의 '내용 구성' 분석 시, 해당 학년군의 성취기준을 참조하여 수업 내용이 교육과정에 적합한지 피드백합니다.
